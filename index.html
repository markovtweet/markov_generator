Markov generator

View project on GitHub

Feature

  • Web based user interface
  • Get lastest tweet of one user
  • Translate it in a weird way

Detail

Source code written in markov.cpp and markov_Co1.cpp, which the first one has the chain length of 2 words and another has 1 word. By searching for the URL with 'https://twitter.com/' + username, the source text is founded. The local executable is called by php and will return the markov style sentence.
